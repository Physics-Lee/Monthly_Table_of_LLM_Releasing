# 北京上海四大AI研究机构 + 国际同类机构

> 本笔记整理北京、上海两地具有代表性的人工智能研究机构，以及全球范围内的同类机构对比。
> 最后更新：2026-04-22

---

## 一、北京智源人工智能研究院（BAAI）

| 项目 | 内容 |
|------|------|
| 全称 | Beijing Academy of Artificial Intelligence |
| 简称 | BAAI、智源研究院 |
| 成立 | **2018年11月** |
| 依托 | 科技部、北京市政府 |
| 地址 | 北京市海淀区成府路150号智源大厦 |
| 官网 | https://www.baai.ac.cn/ |
| 创始人/院长 | 黄铁军（首任） |

### 主要成果

| 系列 | 内容 |
|------|------|
| **悟道**（WuDao） | 中国首个超大规模预训练模型，悟道2.0曾为全球最大（1.75万亿参数） |
| **Emu** | 原生多模态世界模型，Emu3于2024年发布 |
| **BGE** | 文本嵌入模型，Hugging Face 2023年度全球下载量冠军，累计下载超6亿次 |
| **FlagOpen** | 大模型全栈开源技术基座，覆盖数据→训练→评测→推理→系统 |
| **FlagOS** | 智算集群操作系统，支持13种AI芯片 |

### 发展历程

- **2018-11**：研究院成立
- **2019-04**：启动悟道攻关团队
- **2020-10**：悟道1.0发布，首提"大模型"概念
- **2023**：BGE系列成为全球下载量冠军
- **2024-12**：BAAIWorm（天宝）登上 Nature 子刊封面
- **2024-10**：Emu3 发布
- **2025-06**：悟界系列发布
- **2026-03**：多模态大模型成果登 Nature

| 模型 | 说明 |

## 五、国际同类机构

### 1. AI2（Allen Institute for AI）

| 项目 | 内容 |
|------|------|
| 全称 | Allen Institute for AI |
| 简称 | AI2、Ai2 |
| 成立 | **2014年** |
| 创始人 | 保罗·艾伦（微软联合创始人） |
| 地址 | 美国西雅图 |
| 官网 | https://allenai.org/ |

**定位**：与BAAI结构最相似的非营利研究所，专注AI开源透明

#### 代表模型

| 模型 | 发布 | 说明 |
|------|------|------|
| **OLMo** | 2024-02 | 首个全流程开源模型（数据+代码+权重+配方） |
| **OLMo 2** | 2024-11 | 7B/13B，与Qwen 2.5相当 |
| **OLMo 2 32B** | 2025-03 | 首个全开源超越GPT-3.5 Turbo和GPT-4o mini的模型 |
| **OLMoE** | 2024-09 | MoE架构，首个达到Pareto最优的完全开源MoE |
| **Tülu 3** | 2024-11 | 指令微调系列，包含8B/70B/405B |
| **Tülu 3 405B** | 2025 | 全开源后训练方案应用于405B规模 |
| **Molmo** | 2024 | 多模态开源模型 |
| **Molmo 2** | 2025 | 视频-语言模型，开源视频理解能力 |
| **OLMo 3** | 2025-11 | 7B/32B，Base+Instruct+Think+RL Zero完整系列 |
| **OLMo 3.1** | 2025-12 | Think 32B在Artificial Analysis开放性指数排名第一 |

#### 核心数据

- **Dolma**：3T token英文语料库（5.9T for OLMo 3）
- **OLMES**：评测基准
- **Open LLM Leaderboard**：由AI2维护的权威开源模型榜单

#### 与BAAI的相似性

| | BAAI | AI2 |
|---|---|---|
| 性质 | 非营利，政府依托 | 非营利， philanthropist依托 |
| 使命 | AI创新引领+开源生态 | AI for common good+完全透明 |
| 开源 | FlagOpen（数据→推理全链路） | OLMo（全流程开源+模型flow） |
| 旗舰模型 | 悟道/Emu3/BGE | OLMo/Molmo/Tülu |

### 2. EleutherAI Institute

| 项目 | 内容 |
|------|------|
| 全称 | EleutherAI |
| 成立 | **2020年7月**（Discord社群），**2023年**正式注册为非营利研究所 |
| 创始人 | Connor Leahy、 Sid Black、Leo Gao |
| 性质 | 研究合作社（Research Co-operative） |
| 官网 | https://eleuther.ai/ |

**定位**：开源GPT替代品的核心推动者，被誉为"开源版OpenAI"

#### 代表模型/项目

| 项目 | 发布 | 说明 |
|------|------|------|
| **GPT-J** | 2021-06 | 6B参数，开源GPT-3对标，The Pile训练 |
| **GPT-NeoX** | 2022 | 20B参数，GPT-3对标 |
| **The Pile** | 2021 | 825GB高质量数据集，被广泛用于训练开源模型 |
| **Pythia** | 2023 | 8个规模（14M～12B），相同数据和顺序训练，专为可解释性研究设计 |
| **LM Evaluation Harness** | 2020 | 通用LLM评测框架，GitHub 12K+ stars，被Hugging Face广泛使用 |

#### 核心贡献

- Stable Diffusion诞生的关键参与者（The Pile数据支持）
- 推动开源社区打破闭源垄断
- Pythia模型下载超7000万次
- 开源模型贡献于AlphaFold等科学突破

### 4. Stanford HAI

| 项目 | 内容 |
|------|------|
| 全称 | Stanford Institute for Human-Centered Artificial Intelligence |
| 简称 | Stanford HAI |
| 成立 | **2019年** |
| 地址 | 美国斯坦福大学 |
| 官网 | https://hai.stanford.edu/ |

**定位**：不以发布模型为目标，专注AI影响研究、政策与年度AI Index报告

#### 核心产出

| 产出 | 说明 |
|------|------|
| **AI Index Report** | 年度AI行业报告，公认最权威AI发展指标 |
| **AI Index** | 追踪AI性能、投融资、政策、人才培养等维度 |
| 学术研究 | 人本AI基础研究 |
| 产业合作 | 斯坦福校内跨学科AI研究平台 |

**注意**：Stanford HAI本身不发布大模型，主要作为AI行业观察者和评估机构。

## 七、是否加入追踪表的判断

| 机构 | 建议 | 理由 |
|------|------|------|
| BAAI | ✅ 建议加入 | 开源影响力大，Emu3/BGE有辨识度 |
| BIGAI | ❌ 不建议加入 | 偏理论平台，无明确大模型产品发布 |
| 上海AI实验室 | ✅ 建议加入 | InternLM系列有知名度，科学模型有特色 |
| 上海创智 | ❌ 不建议加入 | 发的是框架和科研平台，非大模型产品 |
| **AI2** | ✅ 建议加入 | OLMo系列有影响力，全开源理念独特，与BAAI对应 |
| **EleutherAI** | ✅ 建议加入 | GPT-J/Pythia有历史地位，LM Harness是标配工具 |
| Mila | ⚠️ 谨慎加入 | 偏学术研究，模型产出少，可视情况决定 |
| Stanford HAI | ❌ 不建议加入 | 不发布模型，是评估机构而非发布机构 |
| AI Alliance | ❌ 暂不加入 | 偏联盟组织，非单一机构实体 |

**最值得优先加入**：BAAI、上海AI实验室、AI2、EleutherAI（4个各具特色）
