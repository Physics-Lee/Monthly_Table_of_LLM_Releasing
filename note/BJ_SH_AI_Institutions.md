# 北京上海四大AI研究机构

> 本笔记整理北京、上海两地具有代表性的人工智能研究机构基本信息、产品路线及发布动态。
> 最后更新：2026-04-22

---

## 一、北京智源人工智能研究院（BAAI）

| 项目 | 内容 |
|------|------|
| 全称 | Beijing Academy of Artificial Intelligence |
| 简称 | BAAI、智源研究院 |
| 成立 | **2018年11月** |
| 依托 | 科技部、北京市政府 |
| 地址 | 北京市海淀区成府路150号智源大厦 |
| 官网 | https://www.baai.ac.cn/ |
| 创始人/院长 | 黄铁军（首任） |

### 主要成果

| 系列 | 内容 |
|------|------|
| **悟道**（WuDao） | 中国首个超大规模预训练模型，悟道2.0曾为全球最大（1.75万亿参数） |
| **Emu** | 原生多模态世界模型，Emu3于2024年发布 |
| **BGE** | 文本嵌入模型，Hugging Face 2023年度全球下载量冠军，累计下载超6亿次 |
| **FlagOpen** | 大模型全栈开源技术基座，覆盖数据→训练→评测→推理→系统 |
| **FlagOS** | 智算集群操作系统，支持13种AI芯片 |

### 发展历程

- **2018-11**：研究院成立
- **2019-04**：启动悟道攻关团队
- **2020-10**：悟道1.0发布，首提"大模型"概念
- **2023**：BGE系列成为全球下载量冠军
- **2024-12**：BAAIWorm（天宝）登上 Nature 子刊封面
- **2024-10**：Emu3 发布
- **2025-06**：悟界系列发布
- **2026-03**：多模态大模型成果登 Nature

### 融资与规模

- 非营利机构，资金来源为政府拨款及产业联盟
- 员工约101-250人（含科研、技术、生态团队）
- 开源模型全球累计下载量近6亿次

## 二、北京通用人工智能研究院（BIGAI）

| 项目 | 内容 |
|------|------|
| 全称 | Beijing Institute for General Artificial Intelligence |
| 简称 | BIGAI、通研院 |
| 成立 | **2020年10月** |
| 依托 | 北京市科委、科技部、教育部、北大、清华 |
| 地址 | 北京市海淀区颐和园路2号未来科技大厦 |
| 官网 | https://www.mybigai.ac.cn/ |
| 创始人/院长 | **朱松纯**教授（全球著名计算机视觉、统计与应用数学、AI专家） |

### 六大研究方向

1. 计算机视觉
2. 自然语言
3. 机器学习
4. 认知推理
5. 机器人学
6. 多智能体

### 主要成果

| 成果 | 说明 |
|------|------|
| **通智大脑** | 通用智能体核心系统 |
| **TongAgents（行业智能体工场）** | 面向行业的智能体开发平台 |
| **跨媒体通用人工智能全国重点实验室** | 科技部批准、与北大共建 |
| **AI安全与标准北京市重点实验室** | 与北邮共建 |
| **RoboOS** | 跨本体具身大小脑协作框架 |
| **RoboBrain** | 开源具身大脑 |

### 与BAAI的区别

| | BAAI | BIGAI |
|---|---|---|
| 成立时间 | 2018年 | 2020年 |
| 创始人 | 黄铁军 | **朱松纯** |
| 学术派系 | 北大系偏多 | 清华、北大联合 |
| 核心方向 | 大模型开源生态 | 通用人工智能理论+具身智能 |
| 标志成果 | 悟道、Emu3、BGE | 通智大脑、TongAgents |

## 三、上海人工智能实验室（上海AI实验室）

| 项目 | 内容 |
|------|------|
| 全称 | Shanghai Artificial Intelligence Laboratory |
| 简称 | 上海AI实验室 |
| 成立 | **2021年** |
| 依托 | 上海市政府 |
| 官网 | https://www.shlab.org.cn/ |
| 主任/首席科学家 | 周伯文 |

### 书生大模型家族（InternLM系列）

| 模型 | 类型 | 说明 |
|------|------|------|
| **InternLM** | 大语言模型 | 书生·浦语，千亿参数 |
| **InternVL** | 多模态模型 | 书生·万象 |
| **Intern-S1** | 科学多模态模型 | 万亿参数MoE，科学推理 |
| **Intern-S1-Pro** | 科学多模态（最新） | 全球开源最大科学模型，2026-02发布 |
| **书生·思客** | 强推理模型 | - |

### 核心技术架构

- **SAGE架构**："通专融合"技术架构，实现通用能力与专业能力协同演进
- **DeepLink**：超智融合算力平台，打破超算与智算壁垒
- **Sciverse**：科学智能数据库

### 重要发布

- **2023**：书生大模型首次发布
- **2024**：InternVL（书生·万象）发布
- **2026-02-04**：**Intern-S1-Pro** 开源发布，万亿参数科学多模态大模型
- **2026-04**：发布"珠穆朗玛计划"，全面开放算力/数据/模型/平台合作

### 开源生态

- Intern-S1 多次登顶 HuggingFace 全球多模态榜单
- 累计下载超41万次
- 合作科研机构及企业超200家

## 四、上海创智学院

| 项目 | 内容 |
|------|------|
| 全称 | 上海创智学院（Shanghai Innovation Institute） |
| 简称 | 创智学院、SII |
| 成立 | **2024年** |
| 地址 | 上海市徐汇区华发路699弄3号 |
| 官网 | https://www.sii.edu.cn/ |

### 与上海AI实验室的关系

**完全独立**的两个机构：

| | 上海AI实验室 | 上海创智学院 |
|---|---|---|
| 成立 | 2021年 | 2024年 |
| 定位 | 政府实验室 | 新型研发机构 |
| 依托 | 上海市政府 | 上海市科委、北大、清华等 |
| 核心产出 | 大模型（InternLM） | 框架、智能体、研究平台 |

### 2025年7月首发八大成果

| 领域 | 成果 |
|------|------|
| 认知智能 | Q计划-深度认知引擎 |
| 情境智能 | 拟人化语音交互模型、MOSS-Audio |
| 具身智能 | 万物有灵、RoboSoul |
| 科学智能 | 原子级基座模型（能源化工） |
| AI基础设施 | siiRL（强化学习开源框架）、NEX（Agent全栈方案） |
| 终端模型 | Megrez 2.0（终端本征大模型，3B计算量媲美21B） |
| 教育 | 启创·InnoSpark1.0教育大模型 |
| 评测 | 全球首个AI创造力测评系统 |

### 核心项目

| 项目 | 说明 |
|------|------|
| **siiRL** | 面向下一代推理的强化学习开源框架，100%开源，兼容国产芯片 |
| **NEX** | Agent全栈解决方案（模型+数据+框架+基础设施），nex.sii.edu.cn |
| **科研驾驶舱** | 下一代能动型科研系统操作系统 |
| **BetaWeb** | 全球首个基于区块链的可信智能体互联网 |

## 五、四家机构对比

| 机构 | 成立 | 核心标签 | 代表作 | 开源力度 |
|------|------|----------|--------|----------|
| **BAAI** | 2018 | 大模型开源生态 | 悟道、Emu3、BGE | 极强（FlagOpen） |
| **BIGAI** | 2020 | 通用智能理论 | 通智大脑、TongAgents | 一般 |
| **上海AI实验室** | 2021 | 科学智能 | InternLM、Intern-S1 | 强（Intern系列） |
| **上海创智** | 2024 | 智能体+框架 | siiRL、NEX、Megrez | 强（siiRL全开源） |

## 六、是否加入追踪表的判断

| 机构 | 建议 | 理由 |
|------|------|------|
| BAAI | ✅ **建议加入** | 开源影响力大，Emu3/BGE有辨识度 |
| BIGAI | ❌ 不建议加入 | 偏理论平台，无明确大模型产品发布 |
| 上海AI实验室 | ✅ **建议加入** | InternLM系列已有知名度，科学模型有特色 |
| 上海创智 | ❌ 不建议加入 | 发的是框架和科研平台，非大模型产品 |
