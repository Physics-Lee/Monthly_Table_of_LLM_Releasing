# 表格最后两列说明（草稿）

## 背景

本表格原本只记录**大语言模型（LLM）的发布**，共 32 家模型厂商。

后来新增两列，用于记录**支撑这些模型运行的底层基础设施事件**：

| 列名 | 含义 |
|---|---|
| **LLM-芯片** | 让模型能跑起来的**硬件/组件**的里程碑 |
| **LLM-云计算** | 让模型能被用户用上的**平台/服务**的里程碑 |

---

## LLM-芯片 列

**记录标准**：不是"某个芯片发布了"，而是"没有它，大模型就跑不动/跑不快"的里程碑事件。

**包含四种类型**：

### 1. 核心组件量产

| 例子 | 说明 |
|---|---|
| **SK Hynix HBM3E** | HBM 是 GPU 用的"显存"。HBM3E 带宽提升 50%，是所有高端 AI 芯片（H100、MI300X）的标配。没有它，大模型训练根本跑不动。 |

### 2. AI 芯片发布/上市

| 例子 | 说明 |
|---|---|
| **NVIDIA Blackwell B200** | NVIDIA 下一代旗舰芯片，2024-2025 年所有顶级模型的训练底座。 |
| **AMD MI300X** | 首款真正对标 H100 的芯片，打破 NVIDIA 垄断。 |
| **NVIDIA H200** | H100 的升级款，显存翻倍，让单卡可以运行更大模型。 |

### 3. 云厂商自研芯片

| 例子 | 说明 |
|---|---|
| **AWS Trainium2** | 亚马逊自研训练芯片，客户可以用更便宜的自研芯片训练模型，而不是昂贵的 NVIDIA。 |
| **Microsoft Maia 100/200** | 微软自研推理芯片，用于降低 Copilot 等服务的运行成本。 |
| **Google TPU v5p/v6e** | Google 自研 AI 芯片，只供 Google Cloud 使用，训练 Gemini 系列。 |

### 4. 芯片厂商自己做模型

| 例子 | 说明 |
|---|---|
| **NVIDIA Nemotron 3** | NVIDIA 从"卖铲子的"变成"自己也淘金"，推出自研大模型。 |

---

## LLM-云计算 列

**记录标准**：不是"某个云服务上线了"，而是"让大模型从实验室走向用户"的关键事件。

**包含四种类型**：

### 1. 核心云服务上线

| 例子 | 说明 |
|---|---|
| **Azure OpenAI Service** | 企业不需要自己买 GPU、自己部署模型，直接调用 API 就能用 GPT-4。这是 AI 商业化的起点。 |
| **AWS Bedrock GA** | "模型超市"，一个 AWS 账号可以调用 Claude、Llama、Titan 等多家模型。 |

### 2. 自研芯片商用

| 例子 | 说明 |
|---|---|
| **AWS Trainium2 GA** | 亚马逊自研芯片正式对外提供服务（GA = General Available），标志着云厂商开始摆脱 NVIDIA 依赖。 |
| **AWS AI Factories** | 帮企业在自有数据中心部署 AWS AI 基础设施，从"公有云"扩展到"私有化"。 |

### 3. 融资/投资（说明市场热度）

| 例子 | 说明 |
|---|---|
| **CoreWeave 75亿美元** | GPU 云计算公司 CoreWeave 用 GPU 做抵押，借到 75 亿美元。说明"卖算力"和"做模型"一样值钱，AI 基础设施成为资本最热的赛道。 |
| **Lambda Labs 3.2亿美元** | 另一家 GPU 云公司融资，填补市场空白（AWS 太贵、CoreWeave 太专）。 |
| **Microsoft-OpenAI Stargate 1000亿美元** | 微软和 OpenAI 投资 1000 亿美元建数据中心，人类史上最大的科技基建项目之一。 |

### 4. 算力成本下降（普惠化）

| 例子 | 说明 |
|---|---|
| **AWS GPU降价45%** | NVIDIA GPU 实例降价 45%，算力从"稀缺品"变成"大宗商品"。训练大模型的成本直接减半，对 AI 创业公司是重大利好。 |

## 为什么表格里混着写？

因为**这些事件没有统一的分类标准**：

- 芯片列里有：内存组件、芯片发布、芯片上市、自研芯片、芯片厂商做模型
- 云列里有：服务上线、芯片商用、融资、降价

但它们有一个共同点：**没有这些，大模型就不存在或无法用。**

---

## 举例：GPT-4 为什么能在 2023-03 发布？

```
2022-09  NVIDIA H100 量产          ← 芯片列如果有，会记在这里
2022-11  Azure OpenAI Service      ← 云列
2023-03  GPT-4 发布                ← 模型表
```

**逻辑链**：
1. H100 量产（硬件到位）
2. Azure OpenAI Service（算力平台到位）
3. GPT-4 训练完成并发布（模型到位）
